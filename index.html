What is web development and how it is done

We development is the process of planning, designing, coding and publishing a website on the internet. Almost every website is made with HTML (hypertext markup language). Otherwise they are made with .md markdown language. We will be focusing on HTML for now.


The planning process for developing a website:


  1. A website plan pops up in your head: you see what you can do.
  2. You plan and define the purpose.
  3. You make a final plan.
  4. You decide what software you are going to use to code.
  5. You create the documents and insert media.
  6. You code in html and css.
  7. You add javascript.
  8. You put the website on a url. You can purchase custom urls
  9. You put the website on a public search engine.
  10. You share and showcase your website. Congrats! You have got your very own website!

You can put text, buttons, audio files, video files, images, blocks, forms etc. on your website using HTML.
